# 🌍 SUAN - Application d'Apprentissage du Baoulé

**Préserver et Promouvoir la Langue Baoulé à travers l'Apprentissage Interactif**

[![Status](https://img.shields.io/badge/Status-Production%20Ready-brightgreen.svg)]()

---

## 📱 À Propos

Suan est une application mobile Flutter qui permet aux apprenants d'apprendre la langue Baoulé de manière interactive et ludique.

---

## ✨ Fonctionnalités

- 🔐 Authentification sécurisée avec email/password
- 📚 Leçons structurées par 3 niveaux progressifs
- 🎮 Quiz interactifs avec scoring automatique
- 🏆 Gamification (points, streaks, badges, favoris)
- 👤 Profil utilisateur personnalisé
- 💾 Persistance locale complète avec SharedPreferences
- 🎨 Design moderne Material 3

---

## 🚀 Démarrage Rapide

```bash
# 1. Aller au dossier
```
