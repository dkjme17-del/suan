# Polices pour l'application Suan

Ce dossier contient les polices utilisées pour l'interface en baoulé.

## Polices requises
- Poppins-Regular.ttf
- Poppins-Medium.ttf  
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf

## Note
Les polices Poppins peuvent être téléchargées depuis Google Fonts :
https://fonts.google.com/specimen/Poppins

Pour le développement, vous pouvez utiliser Google Fonts directement dans Flutter
en ajoutant la dépendance `google_fonts` dans pubspec.yaml.
