# Changelog

Tous les changements importants du projet Suan sont documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
et ce projet adhère à la [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-09

### Added
- ✨ Authentification complète (Inscription/Connexion/Logout)
- 📚 Système de leçons par niveau (Débutant, Intermédiaire, Avancé)
- 🎯 Quiz interactifs avec scoring
- 🎮 Mode d'apprentissage multiples (Classique, Ludique, Non-alphabétisé)
- 📊 Gamification (points, séries, favoris)
- 👤 Gestion de profil utilisateur
- ⚙️ Paramètres personnalisables
- 🎨 Design système modern avec Material 3
- 📱 Interface responsive (mobile & tablet)
- 💾 Persistance des données avec SharedPreferences
- 🏗️ Architecture MVVM propre
- 📖 Documentation complète (5 guides)

### Technical
- Flutter 3.11.1+ support
- Dart 3.11.1+ support
- Provider for state management
- Clean Architecture patterns
- 25+ dependencies properly configured
