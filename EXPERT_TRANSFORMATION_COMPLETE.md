## 📋 How to Deploy

### Step 1: Update Navigation
```dart
// main.dart
routes: {
  '/home': (context) => SlidePageRoute(page: HomePage()),
  '/quiz': (context) => FadeScalePageRoute(page: QuizPage()),
  '/settings': (context) => SlidePageRoute(page: SettingsPage()),
},
```

### Step 2: Update Pages
```dart
// home_page.dart
ListView.builder(
  itemBuilder: (context, index) {
    return ModernLessonCard(
      title: lesson.title,
      description: lesson.description,
      level: lesson.level,
      duration: lesson.durationMinutes,
      isFavorite: isFav,
      onTap: () => navigateToLesson(lesson.id),
      onFavoriteTap: () => toggleFavorite(lesson),
      icon: Icons.school,
    );
  },
)
```

### Step 3: Test on Chrome
```bash
flutter run -d chrome
# See smooth animations in real-time
```

### Step 4: Build for Android/iOS
```bash
flutter build apk --release
# OR
flutter build ios --release
```
