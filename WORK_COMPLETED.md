## 🚫 OBSTACLE RENCONTRÉ

### Problème: SSL/TLS Certificate
```
javax.net.ssl.SSLHandshakeException: PKIX path building failed
```

### Cause: Infrastructure
- Gradle try to download deps via HTTPS
- Certificate not recognized by JVM
- **NOT a code problem** ❌

### Preuve que c'est infrastructure:
```
✅ Dart analyze: OK
✅ Imports: OK
✅ Types: OK
✅ Syntax: OK
❌ SSL: Failed (environmental issue)
```

### Solution:
- Lancer sur autre machine
- Configurer proxy/VPN
- Lancer version desktop Windows
- Consulter TROUBLESHOOTING_SSL.md
